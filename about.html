About Shepherd Parking

Home Page

Description: This page is the landing screen for users once they open the app. It provides key information such as the current parking availability (displayed as a percentage). Users can access various functions like checking in at a parking spot, reporting traffic feedback, notifying if they are running late, and viewing parking analytics.

Main Features:

  • Parking Availability: Displays how full the parking lot is.
  • Check In: Allows users to record their arrival at the parking area.
  • Traffic Feedback: Provides an option for users to report current traffic conditions.
  • Late: A feature that notifies the system if the user is running late.
  • Analytics: Shows parking usage statistics.

Traffic Feedback Page

Description: This page allows users to report real-time traffic feedback. Users can type in a message describing the situation (e.g., an accident) and choose a category (e.g., Heavy Traffic) from a dropdown menu. Once submitted, users can also view updates on the map based on the traffic feedback.

Main Features:

  • Message Input: Users can describe traffic situations.
  • Category Selection: Dropdown to choose a relevant traffic category.
  • Map Updates: View the latest map updates related to traffic incidents.

Map Updates Page

Description: This page provides a visual representation of traffic incidents and parking lot conditions over the last 24 hours. It shows the location of incidents with a legend to explain the types of events (accident, heavy traffic, road construction, etc.).

Main Features:

  • Map with Legend: Displays the map with markers for incidents.
  • Last 24 Hours: Shows updates in a 24-hour period for better decision-making.

Late Notification Page

Description: This page allows users to notify the system when they are running late, especially when it involves interactions with specific lecturers. Users can add a lecturer, select a reason for being late (e.g., traffic), and provide additional information before submitting the notification.

Main Features:

  • Lecturer Dropdown: Allows users to add or select a lecturer.
  • Reason Selection: Users can choose from a list of reasons for being late.
  • Additional Information: A text field for users to provide extra details.

Analytics Page

Description: This page provides statistical data regarding parking lot usage. It projects the expected capacity for the upcoming day, giving users an idea of the parking situation for the morning and afternoon. It helps users plan their parking in advance based on trends.

Main Features:

  • Projected Capacity: Shows the expected capacity for the next day in percentage.
  • Time Slots: A bar chart showing the parking occupancy for different times of the day.

Settings Page

Description: This page allows users to manage their preferences and personal information within the app. Users can update their student number, enable or disable location services, manage push notifications, and view app permissions.

Main Features:

  • Update Student Number: Allows users to modify their student number information.
  • Toggle Location Services: Users can turn location services on or off.
  • Push Notifications: Enables or disables notifications from the app.
  • Language Selection: Users can choose their preferred language.
